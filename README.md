### Response Status

- 1xx: Information
- 2xx: Success
  - 200: OK
  - 201: Created
  - 204: No content PUT POST DELETE
- 3xx: Redirection
- 4xx: Client Error
  - 400: Bad Resquest
  - 404: Not Found
- 5xx: Server Error
  - 500: Internal Server Error

Font: https://developer.mozilla.org/en-US/docs/Web/HTTP/Status
